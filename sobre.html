Sobre a KickStream

A KickStream nasceu de uma paixão simples: assistir futebol brasileiro de verdade. Não apenas os grandes jogos da Série A, mas os clássicos estaduais, as rivalidades regionais, os jogos que movem cidades inteiras e que nunca chegam às grandes plataformas.

Fundada em 2025, nossa missão é democratizar o acesso ao futebol nacional — do Gauchão ao Campeonato Baiano, do Cariocão às Eliminatórias para a Copa do Mundo. Toda partida importa para alguém.

Nossa infraestrutura

A plataforma roda 100% na Amazon Web Services (AWS), com arquitetura projetada para suportar picos de acesso nos dias de jogo. Utilizamos:

  • EC2 com Auto Scaling para garantir disponibilidade
  • Application Load Balancer para distribuição de tráfego
  • DynamoDB como banco de dados NoSQL escalável
  • VPC com sub-redes públicas e privadas para segurança

Nossos valores

  • 🏟️ Futebol para todos, não só para quem tem cabo
  • 📡 Transmissões estáveis mesmo nos momentos de maior audiência
  • 🔒 Segurança e privacidade dos nossos usuários
  • 🇧🇷 Valorização do futebol regional brasileiro
